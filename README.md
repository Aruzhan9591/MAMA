# MAMA – Calendar Project in Maple

## Description

This repository contains a university project created for the MAMA course.

The project was implemented in Maple and focuses on working with calendar dates. The main goal of the project is to create a procedure that determines the day of the week for a given date.

The program also correctly handles leap years according to the standard Gregorian calendar rules.

## Main Features

* Calculation of the day of the week for a selected date
* Leap year detection
* Validation of input values
* Support for different month lengths
* Use of Zeller’s congruence algorithm
* Testing on multiple example dates
* Help procedure with a short description of the module

## Technologies Used

* Maple
* Procedural programming
* Calendar algorithms

## Project Structure

* `ABI0008.mw` – Main Maple worksheet with the project implementation

## Example Results

The project was tested on several dates, for example:

```text
19.4.2026 -> Sunday
1.1.2023 -> Sunday
24.12.2024 -> Tuesday
29.2.2020 -> Saturday
```

Leap year tests:

```text
2012 -> true
2100 -> false
2000 -> true
```

## Purpose

The purpose of this project was to practice algorithmic thinking and procedural programming in Maple. The task included creating a working procedure, checking leap years, validating dates, and testing the program on different examples.
